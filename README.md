# ☕ Cafe Project

## 📌 Description
This is a full stack cafe project with frontend and backend.

## 🚀 Features
- User login & register
- Food ordering system
- Responsive UI

## 🛠 Tech Stack
- Frontend: React / HTML / CSS / JS
- Backend: Node.js / Express
- Database: MongoDB

## 📂 Folder Structure
- frontend/
- backend/

## ⚙️ Installation
1. Clone repo:
git clone https://github.com/your-username/cafe-project.git

2. Install dependencies:
cd frontend
npm install

cd backend
npm install

3. Run project:
npm start
